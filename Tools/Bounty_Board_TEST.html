How to Make a JSON file for upload

  1. Create your table in Excel or Google Sheets. (Example) [Template]
    • The bold and underlined words are the column names. They MUST be written exactly as the image.
  2. Go to Mr. Data Converter and make sure the Output is set to JSON - Properties.
  3. Copy your Excel table and paste it into the input box on Mr. Data Converter
  4. Copy the output text and create a new txt file with notepad and paste the output into it.
  5. Save the new file as whatever you want .json. E.G. chicken.json
